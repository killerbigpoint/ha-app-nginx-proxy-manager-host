**Bleeding-edge development channel** — Cutting-edge features, daily builds.

This is the **dev** variant of Nginx Proxy Manager, tracking the `development` branch.
It contains the latest in-progress work and is updated frequently.

## ⚠️ Development Channel Guidelines

⚡ **For:**

- Developers and power users
- Testing specific fixes in development
- Contributing bug reports

🚫 **Not for:**

- Production systems
- Users expecting stability
- Mission-critical deployments

## 🔄 Continuous Updates

Dev builds trigger on every push to `development`. Expect breaking changes, incomplete
features, and frequent updates.

## 🐛 Known Issues & Breaking Changes

Development builds may contain:

- Incomplete features
- Experimental APIs
- Breaking changes
- Performance issues
